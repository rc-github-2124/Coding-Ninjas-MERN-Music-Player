## Run Locally

Clone the project

```bash
  git clone https://github.com/rc-github-2124/Coding-Ninjas-MERN-Music-Player
```

```bash
  cd Coding-Ninjas-MERN-Music-Player
```



Start the server

```bash
  Open with Live Server-(VS Code Extension)
```
